# ToDoApp

ToDoApp es una aplicación de gestión de tareas desarrollada con React. Permite a los usuarios crear, organizar y priorizar tareas dentro de diferentes proyectos, facilitando la gestión del tiempo y la productividad.

## Características:
- Agregar, editar y eliminar tareas.
- Organizar tareas por proyectos.
- Filtrar tareas por fecha (Hoy y Semana).
- Asignar prioridades a las tareas.
- Interfaz simple e intuitiva.

# ToDoApp

ToDoApp is a task management application built with React. It allows users to create, organize, and prioritize tasks within different projects, making time management and productivity easier.

## Features:
- Add, edit, and delete tasks.
- Organize tasks by projects.
- Filter tasks by date (Today and Week).
- Assign task priorities.
- Simple and intuitive interface.
